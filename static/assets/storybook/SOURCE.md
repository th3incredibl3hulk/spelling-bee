# Storybook Quest Assets

This theme is a whimsical fairy-tale spelling adventure for this private family
spelling app. It uses public-domain classic literature art and original local
illustrations.

## Vendored Media

- `Book-icon.svg`
  - Source: `https://upload.wikimedia.org/wikipedia/commons/0/0f/Book-icon.svg`
  - Runtime use: review-only book icon reference.
- `book-emblem.svg`
  - Original local open-book emblem art.
  - Runtime use: Storybook Quest brand mark, fly-in media, collectible token.
